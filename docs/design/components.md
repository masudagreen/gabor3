## GG-1 `GaborGrid`（F-01、v3.1 で 5x5/6x6 拡張）
- n×n のガボールパッチ格子コンテナ。`GaborPatchCell` を n² 個配置。**n は現在レベルの `gridSize`**。**v3.1：既定 3 / 4 に加え、設定で ON にした 5 / 6 にも対応**（system §16.6）。
- 全体辺長：スマホ `min(viewport.short_edge - 32, 360)px`、PC 横 480px（**サイズに依らず一定**。セル数だけ増える）。
- 格子隙間（n 別）：n=3 → `space.2`(8px) / n=4 → 6px / **n=5 → 5px / n=6 → 4px**。クリッピング方式（system §7）により隙間が背景露出の原因にならない。
- パッチ辺長 = (全体辺 − 隙間合計) / n。@328px（360 幅）の目安：n=3 ≈ 104px / n=4 ≈ 77px / **n=5 ≈ 61px / n=6 ≈ 51px**（system §16.6 表）。
- **タップ領域（6x6 の要）**：n=6 でパッチ描画辺 ≈ 51px。**タップ判定領域はパッチ + 周囲隙間半分まで広げ ≈ 55px** とし 48pt 下限を死守（system §16.6）。隙間をこれ以上広げない（視認性維持）。
- **小サイズ視認性（NF-28d）**：n=5/6 でも縞（cpd）・回転/静止弁別が成立すること。破綻時は cpd を下げて縞を太く（Generator 余地、system §16.6/§7.4）。
- **過密回避（§4.7）**：有効個数 < gridSize²（静止が必ず 1 つ以上）。生成時クランプ（`min(count, gridSize²−1)` 目安）。
- 結果開示中は格子全体 `pointer-events: none`（締め切り後の誤操作防止、F-03）。
- a11y: `role="group"` + aria-label「回転しているパッチをすべて選んでください」。各セルは `GaborPatchCell` が `role="checkbox"`。n=6 でも Web は Tab で 36 セル全到達。

## GG-2 `GaborPatchCell`（F-01）
- 1 個のガボールパッチ。中性グレー #808080 背景 + 正弦波縞（system §7 クリッピング）。
- **直接タップでトグル選択**。選択中は控えめな枠（`color.selection.subtle` 線幅 2px、`radius.sm`）。未選択は枠なし or `color.selection.subtle.idle` 1px。**ガボール本体の縞の視認を阻害しない**（枠は半透明・細い、F-01 受け入れ基準）。
- 回転アニメは課題上必須（reduced-motion でも止めない、NF-13）。回転には 3 種の挙動：
  - **一方向回転**：現在レベルの `rotationSpeed`（deg/sec）で一定方向に連続回転（system §4.1）。
  - **振動（往復回転）**：同じ角速度で時計回り↔反時計回りを往復（system §4.1 のモーション狙い）。
  - **静止**：ランダムな初期角度で固定（互いに 12° 以上分散、system §7.2、時間変化なし）。
- **回転の種類（一方向/振動）や速度をユーザーが指定する UI は持たない**（F-01 受け入れ基準）。「現在の回答」テキストも出さない。
- a11y: `role="checkbox"` + `aria-checked`（NF-15）、`aria-label="パッチ {行}-{列}"`。Web は Tab 到達 + Space トグル。
- 結果開示時は `OV-2 ResultMark` を子レイヤとして重畳（中央配置、`color.result.overlayBg` 円の上にアイコン）。**v3.1：5x5/6x6 の小パッチでは選択枠 2px・overlayBg 円（直径 ≒ パッチ辺 55〜60%）・✅/❌ アイコンを比例縮小**（6x6 辺 51px → 円 ≈ 28px、アイコン ≈ 20px。縞を完全には覆い隠さない透過度を維持、F-03/system §16.6）。

## LB-1 `LevelBadge`（F-02、新規）
- 現在のレベル番号を示すピル/タイル。ゲーム上部・ホーム結果・履歴で共用。
- バリアント：
  - `inline`（ゲーム上部バー、GB-1 内）：`font.label` 24px Bold tabular-nums、「レベル {n}」。`color.level.bg` ピル + `color.level.fg` テキスト（不透明、7:1）。
  - `large`（ホーム結果カード）：`font.display` 64px Bold + 上に「現在のレベル」`font.label`。
  - `compact`（履歴 StatTile 内/グラフ凡例）：`font.numeric.l` 48px。
- レベル番号は最大 3 桁（720）想定で桁ぶれしない幅（tabular-nums）。
