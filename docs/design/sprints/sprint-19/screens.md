### 教示表示の差分仕様（F-02/AS-36）
| 項目 | 旧（v3.0/v3.1・本番） | 新（v3.2・本番） |
|---|---|---|
| **位置** | ゲーム上部バー直下・格子の上（CB-1） | **同一**（位置変更なし、CB-1 のまま） |
| **文言** | 「{n} 個探せ！」（`game.count.findN`） | **「全て探せ」**（`game.find.all`） |
| **バリアント** | CB-1 `count`（個数明示） | **CB-1 `all`（個数非表示）** |
| **フォントサイズ** | `font.body.lg` 26px Bold | **同一**（`font.body.lg` 26px Bold、18pt 以上を維持） |
| **色 / 背景** | `color.fg.primary` on 不透明ピル `color.bg.surface`（7:1） | **同一** |
| **個数表示** | あり（レベルが決定） | **なし**（個数は難易度軸から除外、各ラウンドでランダム＝内部値） |

- **CB-1 `all` バリアント**：文言「全て探せ」固定。**ゲーム中・ラウンド中に変動しない**（F-02 受け入れ基準）。回転個数がラウンドごとにランダムでも、教示文言は常に「全て探せ」で不変。
- **回転種類/速度/個数を指定する UI はゲーム画面に存在しない**（F-01 受け入れ基準を維持）。個数も「数当て」ではなく「全部見つける」課題に統一（AS-36）。
- **個数のランダム生成（内部・画面非表示）**：各ラウンドで回転パッチ数を `countRange`（§4.9・SR-1c）の範囲からランダム決定。格子容量クランプ（§4.7、回転個数 < gridSize²、静止が必ず 1 つ以上残る）を適用。これらは内部仕様で、**画面に数値・範囲を一切表示しない**（F-03「内部数値テキストは表示しない」と整合）。

### a11y（F-02/NF-10、v3.2）
- ゲーム開始時に `aria-live="assertive"` で「**回転しているものを全て探してください**」（`game.a11y.start.all`）。**個数は読み上げない**（本番は個数非表示）。旧「レベル {level}。{count} 個…」（`game.a11y.start`）から置換。
- レベル番号は GB-1 右ピルで読み上げ（aria-label「レベル {n}」、不変）。
- CB-1 `all` の aria-label「回転しているものを全て探してください」。
- `GaborGrid` `role="group"` aria-label「回転しているパッチをすべて選んでください」（既存・不変）。
- その他（パッチ選択・カウントダウン・X 中断・結果開示の読み上げ）は S5（sprint-5）から不変。

### 結果開示（S19-2 ≒ S5-2、F-03）
- **結果開示（✅/❌・総合クリア/失敗・3 秒カウントダウン）は sprint-5 の S5-2 から不変**。個数を出さないことと整合（元々開示にも個数の内部数値は出さない）。
- クリア判定は「回転していたパッチを過不足なく全選択」（§4.3、不変）。個数明示がなくても、ユーザーは「全部見つけたか」を ✅/❌ 開示で確認できる。
- 教示バナー（CB-1 `all`「全て探せ」）は開示中も残してよい（S5-2 で CountBanner を残すのと同様、参照のため）。

### レスポンシブ
- S5（sprint-5）から不変。360/375/1280 で CB-1「全て探せ」は短文のため折り返さず収まる（旧「{n} 個探せ！」より短い or 同等）。
