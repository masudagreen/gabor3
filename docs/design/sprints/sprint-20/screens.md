> **既定 `repeatCount=4` ＋ `defaultVariableRanges`（外側のみ・v3.1 同一）→ 総レベル数 4×180=720**（v3.1 と一致）。`repeatCount` を変えると総レベル数（n×180）が変わる。`countRange` は総レベル数を変えない（個数は難易度軸ではない、§4.9）。

### 状態遷移（F-13、v3.2）

#### `repeatCount`（SR-1b、★梯子に影響）
- [−]/[+] で 1〜6 回を 1 刻み増減（1 で [−] disabled、6 で [+] disabled）。即座に `Settings.repeatCount` 保存。
- **梯子への波及（§4.5）**：総レベル数が n×180 に変わるため、`sessionMinutes` とは異なり**現在レベルを新有効範囲 [1, n×180] にクランプ + 連続失敗カウント 0 リセット**（RG-1/OR-1 と同じ波及）。総レベル数プレビューを即更新。クランプ発生時は予告/事後トースト「レベルの梯子が変わります。現在レベルは {clamped} に調整されます」（`font.caption`）。
  - 補足：既定 4 から増やす（5,6）と総レベル数は増えるのみ（720→900→1080）でクランプは通常起きない。減らす（3,2,1）と総レベル数が減り（540/360/180）、現在レベルが新上限超過ならクランプが発生する。

#### `countRange`（SR-1c、★梯子に影響しない）
- 3 択ラジオ（FT-3）で 1 つ選択。即座に `Settings.countRange` 保存、**次ラウンドから本番の個数ランダム範囲に反映**。
- **梯子への波及なし**（個数は難易度軸ではない、§4.9）。`sessionMinutes` と同様、クランプ・連続失敗リセットは**発生しない**。総レベル数プレビューも不変。
- 各プリセットの意味を `font.caption` で 1 行（「セル数いっぱい＝1〜(セル数−1)」「ひかえめ＝1〜半分」「1〜4 固定」）。内部の格子容量クランプ（回転個数 < gridSize²）は UI に出さない（ユーザー非表示の内部仕様）。

#### RG-1（範囲設定、v3.2：個数除去）
- 個数チップ群を表示しない。時間 / 回転方向 / サイズ / 回転速度の 4 変数のみ。挙動（追加値 OFF・flex-wrap 折返し・1 値以上必須・変更で梯子クランプ）は S3 から不変。

#### OR-1（変化順、v3.2：個数除去）
- 並べ替え対象は外側 4 変数（時間 / 回転方向 / サイズ / 回転速度）のみ。「個数」行を表示しない。
- 「繰り返し（最内側・固定）」を**並べ替え不可の固定行**として最上段に表示してよい（ユーザーに最内側が繰り返しと分かるように。`color.fg.muted` グレーアウト・上下ボタンなし）。または固定行を省き caption「変化順は繰り返しの外側の 4 変数のみ調整します」で代替（Generator 判断）。
- デフォルト順：繰り返し（固定）→ 時間 → 回転方向 → サイズ → 回転速度。組み替えで梯子が変わる → クランプ + 連続失敗 0 リセット（不変）。

### 状態：エンプティ/ロード/エラー
- 初回（設定未保存）：`repeatCount=4` / `countRange=half` / 範囲はフル既定 ON / 変化順デフォルト。
- マイグレーション（v3.1 → v3.2、§7.9）：既存 `Settings` に `repeatCount`（既定 4）・`countRange`（既定 half）を補完。`variableRanges`/`variableOrder` から `count` 要素を除去（存在しても無視）。`repeatCount=4` なら総レベル数 720 維持で `currentLevel` クランプ不要。
- ロード：永続化読込中は短時間スケルトン or 既定値で即表示（NF-3）。
- エラー状態は実質発生しない（チップ/ラジオは値集合内のみ、ステッパーは下限上限で構造的に止まる）。

### a11y（v3.2）
- SR-1b（repeatCount）：`role="spinbutton"` `aria-valuenow={n}` `aria-valuemin=1` `aria-valuemax=6` `aria-valuetext="{n}回"`、[−][+] 各 aria-label「1 回増やす/減らす」。値変更・クランプを `aria-live="polite"` で読み上げ。
- SR-1c（countRange）：`role="radiogroup"` aria-label「個数の範囲」、各 `role="radio"` + `aria-checked`、各ラジオの説明を `aria-describedby`。
- RG-1：各チップ `role="checkbox"`（個数チップは存在しない）、グループ aria-label「{変数名}の範囲」。
- OR-1：`role="list"`、組み替え可能 4 行は `role="listitem"` + 上下ボタン aria-label、繰り返し固定行は `aria-disabled="true"` aria-label「繰り返し（最内側・固定）」。
- その他（タイトル h1 を Skip link 着地点、Toggle=switch、全削除ダイアログのフォーカストラップ）は S3 から不変。
