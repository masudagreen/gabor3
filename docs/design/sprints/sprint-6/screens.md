## 起動フロー全体（F-06/AS-6）

```
アプリ起動
  → 旧データ検出時：DataResetNotice（S2-2、1 度のみ）→ OK
  → 初回（onboardingCompleted=false）：オンボーディング 4 ステップ（S6-1）
  → 距離リマインド（S6-2、自動進行）
  → ホームタブでゲーム自動開始（S4 プレイ）
  → r ラウンド完了 → ホームにセッション結果（S6-3）
  → 「もう一度」→ 距離リマインド → 新セッション
```
- クールダウン画面なし（廃止、F-06）。
- 2 回目以降：オンボなし → 距離リマインド → 自動開始。

## 画面 S6-1：オンボーディング（初回のみ、F-06/F-10）

`ON-1 OnboardingStep`。4 ステップ、**合計タップ 6 以下**。セーフエリア準拠。

### ステップ構成
```
[1/4] 免責同意（F-10/DC-1）
┌─────────────────────────────┐
│ GaborEye へようこそ           │ ← font.h1 36px
│ ┌─────────────────────────┐ │
│ │ 本アプリは医療機器では   │ │ ← DisclaimerPanel（薄黄背景）
│ │ ありません。自助セルフ   │ │   font.body 24px 以上
│ │ ケア目的です。対象外：    │ │   対象外ユーザー明記
│ │ お子さま/70 代以上/重度  │ │
│ │ 視覚障害/強度近視/目に    │ │
│ │ 違和感のある方           │ │
│ └─────────────────────────┘ │
│ ☐ 内容を理解しました        │ ← チェック（タップ1）
│ ┌─────────────────────────┐ │
│ │      同意する            │ │ ← primary、未チェックで disabled（タップ2）
│ └─────────────────────────┘ │
│ ● ○ ○ ○                     │ ← 進捗ドット
└─────────────────────────────┘

[2/4] 年齢層（任意）
  「年齢層を選んでください（任意）」
  [40代][50代][60代][70代以上][未回答]  ← Segmented/縦リスト（タップ3）
  → 70代以上選択時：追加警告「視覚に不安がある場合は医師にご相談ください」表示

[3/4] 視聴距離
  「視聴距離を選んでください」
  [30cm][40cm][50cm]  ← SegmentedControl 既定40（タップ4）

[4/4] ゲーム概要（ごく短い）
  「ゆっくり変化（回転・周波数）するパッチを見つけてタップ」
  小さな静止/動くパッチの図解 1 枚（静止 SVG、点滅なし）
  ┌──────────────────────┐
  │      はじめる          │ ← primary（タップ5）→ 距離リマインドへ
  └──────────────────────┘
```

### 規範（F-06/F-10）
- 初回のみ。`onboardingCompleted=true` 後は表示しない。
- 免責同意未チェックでは「同意する」disabled（先に進めない）。同意日時を `disclaimerAgreedAt` に保存。
- 70 代以上申告時：追加警告 + 医師相談推奨（F-10）。年齢層は `ageGroup` に保存（任意=unspecified 可）。
- 合計タップ：理解チェック1 + 同意1 + 年齢1 + 距離1 + はじめる1 = 5（≤6、F-06）。各ステップ「次へ」を別ボタンにしない（選択=自動進行 or 同一ボタンで集約してタップ数を抑える。Generator 判断、ただし 6 以下厳守）。
- 各画面 1 アクション、主要 CTA 1 個（OPT-3）。

### a11y
- 進捗を aria（「ステップ 1/4」）。免責はスクロール可。チェックは `role="checkbox"`。
- 全ボタン 56pt 以上、focus ring。

## 画面 S6-2：距離リマインド（F-06/DR-1）

セーフエリア準拠。短いカウントダウンで自動進行。

### レイアウト
```
┌─────────────────────────────┐ ← セーフエリア
│  ✕                          │ ← 緊急脱出のみ
│                              │
│     画面から 40cm            │ ← font.h1 36px Bold
│      離れてください           │
│                              │
│  （片眼 ON 時）              │
│   片目を覆ってください（任意） │ ← font.h3 26px
│                              │
│         ┌──────┐            │
│         │  3   │            │ ← CountdownTimer large 72px（赤・danger）
│         └──────┘            │
│    3 秒後に自動で始まります   │ ← font.body 24px
└─────────────────────────────┘
```

### 規範（F-06/OPT-10/F-12）
- 「画面から {n}cm 離れてください」を 18pt 以上（実際 36px）で表示（{n}=`viewingDistanceCm`）。
- カウントダウン（Designer 提案 **3 秒**）で自動進行 → ゲーム開始。ユーザー操作（X 中断以外）不要。
- カウントダウンは F-12 統一仕様（数字のみ・段階色：3 秒なので開始から赤 danger・Black 太字・点滅なし・ティック音は S9）。
- 片眼ガイダンス（off 以外）時のみ「片目を覆ってください（任意）」を表示。交互時は左右の指示も（Generator 判断）。
- クールダウン画面なし（F-06）。

### a11y
- カウントダウン aria-live（≤3 秒で毎秒読み上げ）。距離数値読み上げ。X = aria-label「中断」。

## 画面 S6-3：ホームタブ・セッション結果（F-08/F-04）

r ラウンド完了後、ホームタブに `RC-1 SessionResultCard` を表示（セーフエリア準拠、NF-30）。ゲーム非進行状態 → タブ移動自由。

### レイアウト
```
┌─────────────────────────────┐ ← セーフエリア
│        今回のスコア           │ ← font.label 24px
│         ┌─────────┐          │
│         │   72    │          │ ← font.numeric.xl 72px Bold ／100
│         │  ／100  │          │
│         └─────────┘          │
│      🔥 連続 5 日             │ ← ストリーク（炎 + テキスト、色非依存）
│                              │
│  （バッジ獲得時：BadgeAwardToast を 1.5 秒）│
│                              │
│  ┌─────────────────────────┐ │
│  │       もう一度           │ │ ← primary lg 64px（≥56pt、F-08）
│  └─────────────────────────┘ │
├─────────────────────────────┤
│  [● ホーム] [履歴] [設定]    │ ← BottomTabBar
└─────────────────────────────┘
```

### 規範（F-08/F-04）
- スコア 0〜100 を最大強調（72px tabular-nums）。`SessionRecord.sessionScore`。
- 今日のストリーク（`Streak.currentStreak`）表示。
- 「もう一度」56pt 以上（実際 64px）。回数制限なし。押すと**距離リマインド経由で**新セッション（F-08）。
- 結果表示中はゲーム非進行 → タブ移動自由（F-05/F-07）。
- ホームタブの状態は 2 つ：①ゲームプレイ中（S4）/ ②セッション結果（本画面）。起動直後（距離リマインド後）は①。

### a11y
- `role="region"` aria-label「セッション結果。スコア {n} 点（100 点満点）。連続 {n} 日」。
- 「もう一度」focus ring、56pt 以上。バッジ演出は点滅なし。
