```
┌─────────────────────────────────────┐

│  │                                 ││
│  │  ┌─────────────────────────┐   ││
│  │  │                         │   ││ ← TextField 56px
│  │  └─────────────────────────┘   ││
│  │                                 ││
│  │  ┌─────────────────────────┐   ││

│  │                                 ││

│  │  └─────────────────────────┘   ││   テキストが「削除」と一致時のみ enabled
│  │                                 ││
│  └─────────────────────────────────┘│
│                                     │
└─────────────────────────────────────┘
```

### 動作
- TextField に「削除」と完全一致した文字を入力 → 「削除」ボタン enabled
- それ以外は disabled
- 「削除」タップ → 全データ削除実行 → オンボーディング S4-01 へ
- 「キャンセル」→ S7-04 / S7-01

### 削除対象（spec.md §10.2）
- UserProfile / Settings / StaircaseState / SessionRecord / TrialRecord / DailyStats / Streak / BadgeStatus すべて
- 削除後はオンボーディング画面からやり直し

### a11y
- TextField `aria-label="確認のため『削除』と入力"`
- 「削除」disabled 時 `aria-disabled="true"`、ヒント「『削除』と入力するとボタンが有効になります」
