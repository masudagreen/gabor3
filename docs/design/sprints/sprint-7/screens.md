## 起動フロー全体（F-06/AS-6/AS-15）

```
アプリ起動
  → 旧データ（v1〜v2）検出時：DataResetNotice（S3-2、1 度のみ）→ OK
  → 初回（onboardingCompleted=false）：オンボーディング 4 ステップ（S7-1）
  → 距離リマインド（S7-2、自動進行）
  → ホームタブで現在レベルの【セッション】自動開始（S5：ラウンド反復）
       ・初回/全削除後：L1 から（AS-15）
       ・2 回目以降：前回の currentLevel から
  → セッション中：ラウンド → 開示(3秒) → 次ラウンド … を sessionMinutes×60 秒を超えるまで反復（S5-1↔S5-2）
  → セッション終了（指定時間超過）→ ホームに【セッション要約】カード（S7-3）
  → 「もう一度」→ 距離リマインド → 新しいセッション（増減後の currentLevel、累計プレイ +1）
```
- クールダウン画面なし（廃止、F-06）。眼精疲労クールダウンも存在しない。
- 2 回目以降：オンボなし → 距離リマインド → 自動開始。
- **v3.1 の変更**：1 ゲーム＝1 ラウンド → **セッション（指定時間までラウンド反復）**。ラウンドごとの結果開示はゲーム画面内（S5-2）、ホームには**セッション末に要約**（S7-3）を表示する（両者は別物、F-08）。

### 規範（F-06/F-10）
- 初回のみ。`onboardingCompleted=true` 後は表示しない。

- 70 代以上申告時：追加警告 + 医師相談推奨（F-10）。年齢層は `ageGroup` に保存（任意=unspecified 可）。
- 合計タップ：理解チェック1 + 同意1 + 年齢1 + 距離1 + はじめる1 = 5（≤6、F-06）。
- 各画面 1 アクション、主要 CTA 1 個（OPT-3）。
- **概要文を「回転」のみに**（v2 の「回転・周波数」から空間周波数の言及を削除、回転のみが変化軸、spec §0）。

### a11y
- 進捗を aria（「ステップ 1/4」）。免責はスクロール可。チェックは `role="checkbox"`。
- 全ボタン 56pt 以上、focus ring。

## 画面 S7-2：距離リマインド（F-06/DR-1）

セーフエリア準拠。短いカウントダウンで自動進行。v2.0 をそのまま流用。

### レイアウト
```
┌─────────────────────────────┐ ← セーフエリア
│  ✕                          │ ← 緊急脱出のみ
│                              │
│     画面から 40cm            │ ← font.h1 36px Bold
│      離れてください           │
│                              │
│  （片眼 ON 時）              │
│   片目を覆ってください（任意） │ ← font.h3 26px
│                              │
│         ┌──────┐            │
│         │  3   │            │ ← CountdownTimer large 72px（赤・danger）
│         └──────┘            │
│    3 秒後に自動で始まります   │ ← font.body 24px
└─────────────────────────────┘
```

### 規範（F-06/OPT-10/F-12）
- 「画面から {n}cm 離れてください」を 18pt 以上（実際 36px）で表示（{n}=`viewingDistanceCm`）。
- カウントダウン（Designer 提案 **3 秒**）で自動進行 → ゲーム開始。ユーザー操作（X 中断以外）不要。
- カウントダウンは F-12 統一仕様（数字のみ・段階色：3 秒なので開始から赤 danger・Black 太字・点滅なし・ティック音は S10）。
- 片眼ガイダンス（off 以外）時のみ「片目を覆ってください（任意）」を表示。交互時は左右の指示も（Generator 判断）。
- クールダウン画面なし（F-06）。

### a11y
- カウントダウン aria-live（≤3 秒で毎秒読み上げ）。距離数値読み上げ。X = aria-label「中断」。

## 画面 S7-3：ホームタブ・セッション要約（F-08/F-04、v3.1 改訂）

セッション終了（指定時間超過でラウンド反復終了）後、ホームタブに `RC-1 SessionSummaryCard` を表示（セーフエリア準拠、NF-30）。ゲーム非進行状態 → タブ移動自由。**v3.1：1 ラウンド結果ではなく、セッション全体の集計（ラウンド数 / クリア・失敗数 / 現在レベル / 今セッション最高 / 今日のストリーク）を表示**（system §16.4、AS-29）。1 ラウンドごとのクリア/失敗・レベル変化はゲーム画面内開示（S5-2）で済んでいる。

### レイアウト（例：5 ラウンド完了、クリア 3 / 失敗 2、開始 L21 → 終了 L23、今セッション最高 L24）
```
┌─────────────────────────────┐ ← セーフエリア
│        セッション終了         │ ← font.h2 30px（または「おつかれさまでした」）
│                              │
│        現在のレベル           │ ← LB-1 LevelBadge large（主役・最大）
│         ┌─────────┐          │   font.label「現在のレベル」
│         │   23    │          │   + font.display 64px（= 次セッション開始レベル）
│         └─────────┘          │
│                              │
│   このセッションの最高  24    │ ← font.numeric.l 48px + ラベル（一回り小さく）
│                              │
│  ┌────────┬────────┬────────┐ │ ← ST-1 compact 3 タイル（ラウンド集計）
│  │ラウンド │ クリア │ 失敗   │ │   各 font.body.lg 26px Bold 数値
│  │   5    │   3    │   2    │ │   クリア=緑アクセント、失敗=中立（責めない）
│  └────────┴────────┴────────┘ │
│                              │
│      🔥 連続 5 日             │ ← ストリーク（炎 + テキスト、色非依存）
│                              │
│  （バッジ獲得時：BadgeAwardToast を 1 度）│
│                              │
│  ┌─────────────────────────┐ │
│  │       もう一度           │ │ ← primary lg 64px（≥56pt、F-08）
│  └─────────────────────────┘ │
├─────────────────────────────┤
│  [● ホーム] [履歴] [設定]    │ ← BottomTabBar
└─────────────────────────────┘
```

### 規範（F-08/F-04、v3.1）
- **見出し**：「セッション終了」`font.h2` 30px（`session.summary.title`。OPT-6 で「おつかれさまでした」も可）。
- **現在のレベル（主役・LB-1 large）**：増減反映後の `LevelState.currentLevel`（= 次セッション開始レベル）を 64px で最大強調。`color.level.fg`。
- **このセッションの最高（`SessionRecord.highestLevelInSession`）**：`font.numeric.l` 48px + `font.label`。主役より一回り小さく。
- **ラウンド集計（ST-1 compact 3 タイル）**：「ラウンド {roundCount}」「クリア {clearCount}」「失敗 {failCount}」。各 `font.body.lg` 26px Bold 数値 + ラベル。クリアは緑アクセント、失敗は中立（赤を主張しない、OPT-6）。360px で 3 列、収まらなければ 2+1 折返し。
- **今日のストリーク**：`Streak.currentStreak`（🔥 + テキスト、色非依存）。
- **LD-1（ラウンド昇降告知）は本カードに出さない**（ゲーム画面内開示 S5-2 へ移設、system §16.3/§16.4）。スコア数値（0〜100）も廃止。
- 「もう一度」56pt 以上（実際 64px）。**回数制限なし**。押すと**距離リマインド経由で新しいセッション**（増減後の currentLevel から、累計プレイ回数 +1、F-08）。
- セッション要約表示中はゲーム非進行 → タブ移動自由（F-05/F-07）。
- ホームタブの状態は 2 つ：① セッションプレイ中（S5）/ ② セッション要約（本画面）。起動直後（距離リマインド後）は ①。

### a11y
- `role="region"` aria-label「セッション終了。現在のレベル {n}。このセッションの最高 {h}。ラウンド {r}、クリア {c}、失敗 {f}。連続 {d} 日」。
- 「もう一度」focus ring、56pt 以上。バッジ演出は点滅なし。

### レスポンシブ
- 360/375：1 カラム、ラウンド集計 3 タイルは 3 列（収まらなければ 2+1）。1280：最大 720px 中央寄せ。
