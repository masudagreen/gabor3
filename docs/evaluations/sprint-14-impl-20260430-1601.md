## 発見した問題

### [Minor] SideBySideStimulus の testId prop が直接子セルに伝播していない

**観察**：G-06 の SideBySideStimulus に `testId="g06-stimulus"` を渡しているが、内部の左右パッチセルの data-testid は `g02-stimulus-left` / `g02-stimulus-right` のままで、`g06-stimulus-left` のような派生命名にはなっていない（コンポーネント default の `g02-...` が常に使われる）。

**影響**：機能・a11y には影響なし。テストの読みやすさのみ。Sprint 13 でも同型（G-05 でも同様の挙動）であり、Sprint 13 評価でも合格としていたため、本評価でも合格事由とする。

**修正の方向性**（任意）：`SideBySideStimulus` の内部で testId prop を受け取り、`testId-left` / `testId-right` 派生 ID を子セルに付与する。本 Sprint で対応する必要なし。
