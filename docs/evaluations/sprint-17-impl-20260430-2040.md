## 発見した問題

### [Minor] 既存フレームワーク警告（本スプリント由来ではない）

ブラウザコンソールに以下警告が出るが、本スプリントの実装変更とは無関係:

- `props.pointerEvents is deprecated. Use style.pointerEvents`
- `Animated: useNativeDriver is not supported because the native animated module is missing`
- `"textShadow*" style props are deprecated. Use "textShadow"`

これらは React Native Web ライブラリ側の deprecation 警告で、Sprint 7 / 14 など過去スプリントから持ち越された既知事象。本スプリント評価では合否に影響しない。Sprint 18 以降のメンテで一括対応推奨。

### [Note] G-13 結果画面の数字 2 重表示（懸念だが基準満たす）

G-13 結果画面の bodyText に「33」（正解の数字 3 が 2 回連続）が観測された。実装的には highlightDigit オーバーレイが正本のコントラストで描画されるため、ノイズ背景 Text + highlight Text の 2 重描画と思われる。スクリーンショット未取得（時間制約）だが、AC「数字を本来コントラストで描画」は満たす。視覚上の重複は self-review §3.4 の設計通り（プレイ中は半透明、採点後は本来コントラスト）。問題なしと判断。
