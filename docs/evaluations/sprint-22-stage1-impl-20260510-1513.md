- ファイル: `src/screens/v11/Onboarding/OB01Welcome.tsx:46`

  2. キャッチコピーに「60 秒の注視で目を鍛える 視覚トレーニング 13 種」と表示される
- 期待: v1.2 では 7 ゲーム。「視覚トレーニング 7 種」に修正
- 実際: 13 種のまま（v1.1.4 の chore コミット e83d67e が UI 文言を整理した際の取りこぼし）
