## 発見した問題
### [Minor] バッジセルの aria-expanded が Web DOM に反映されない
- 再現手順: 履歴タブでバッジセルをタップ → 条件全文が展開される。DOM 上の role=button 要素に `aria-expanded` 属性が出ない（null）
- 期待される振る舞い: accessibilityState.expanded が aria-expanded として SR に伝わる
- 実際の振る舞い: 展開自体は機能し条件全文は読み上げ可能だが、開閉状態が aria-expanded で通知されない（RN Web の accessibilityState シリアライズ挙動）
- 該当箇所: src/components/v2/BadgeCell.tsx（accessibilityState={{ expanded }}）
- 修正の方向性: 必須ではない（展開内容そのものは SR で読める）。厳密化するなら Web で aria-expanded を明示付与。S10 a11y 仕上げで対応可
